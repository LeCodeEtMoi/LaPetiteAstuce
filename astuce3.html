Quels que commande

Explication des commande de façon imager Imaginez que vous avez plusieurs objets dans votre maison que vous voulez envoyer à un ami. Cette commande est comme une boîte vide dans laquelle vous mettez tous ces objets. Elle prépare tous les fichiers modifiés et nouveaux pour être envoyés dans un "colis" de version (commit).
git add .
Cette commande est comme la fermeture de la boîte que vous avez remplie avec les objets. Vous fermez la boîte et collez une étiquette sur laquelle vous écrivez ce que contient le colis (dans ce cas, "Initial commit" est le commentaire qui décrit les changements que vous avez effectués dans ce colis).
git commit -m "Initial commit"
Maintenant que vous avez préparé et fermé la boîte avec tous les objets, vous pouvez l'envoyer à votre ami. Cette commande envoie le colis (commit) que vous avez créé vers votre dépôt distant sur GitHub (dans ce cas, le dépôt s'appelle "origin" et la branche principale s'appelle "main").
git push origine main
Voilà normalement tout est bon
