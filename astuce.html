Étape 1 Comment créé un dépot Gitub
Il faut donc se connecter sur Github .
Une fois connecté sur Github aller sur le + en haut puis aller sur New repository .
Choissir le nom de votre projet valider . Puis faire la commande suivante :
git clone https://github.com/mon/dépot/

Étape 2 Créé une clef SSH

Pour créé un clef SSH réaliser la commande suivante
ssh-keygen -t rsa -b 4096 -C "exemple@mail.com"
Suivre le proces et coppier le contenue de la clef
Aller sur Github > settings >SSH and GPG keys > puis généré une clef

Étape 3 Quelques que commande sur le terminal

Voici les commande à faire .
La commande n'est pas obligatoire mais elle est préférable pour un projet à long terme
git branch -M main
Les autre autre sont importante
git add .

git push origine main
