Étape 1 : Se conecter a son compte Github .

Cliquer sur le bouton + en à gauche puis sur "New répositorie"
Donnez-lui un nom approprié et définissez les options de visibilité (public ou private) selon vos besoins.
Initialisez votre dépôt : Vous pouvez choisir d'initialiser votre dépôt avec un fichier README.md et/ou un fichier.gitignore.
Ces fichiers peuvent être utiles pour fournir des informations sur votre projet et ignorer certains fichiers dans le suivi de version.

Étape 2 : Clonez le dépôt localement

Sur votre ordinateur, ouvrez une interface en ligne de commande (terminal) et naviguez vers l'emplacement où vous souhaitez cloner le dépôt.
Pour rappel on utilise la commande cd pour ce déplacer .
Utilisez la commande git clone suivie de l'URL de votre dépôt pour le cloner localement.
Exemple : git clone https://github.com/votre-utilisateur/votre-depot.git

Maintenant que vous l'avez clonnez ajouter vos fichier . Puis les commande dans l'orde sans rien et avec la bonne orthographe
  • git add .
  • git commit -m "Votre message de commit"
  • git push origin main

    • Il est possible que la branche mère n'est pas forcement main
  • git remote add origin URL_du_dépôt distant
    • Assurez-vous de remplacer URL_du_dépôt distant par l'URL réelle de votre dépôt distant.
  • git remote add origin https://github.com/exemple
  • git branch -M main

    Rendez-vous sur votre compte GitHub, cliquez sur votre avatar en haut à droite et sélectionnez "Settings" (Paramètres).
    Dans le menu de gauche tout en bas , sélectionnez "Developer settings" (Paramètres du développeur), puis "Personal access tokens" (Jetons d'accès personnels).
    Cliquez sur le bouton "Generate new token" (Générer un nouveau jeton).
    Donnez un nom à votre jeton, sélectionnez les autorisations nécessaires (probablement "repo" pour accéder aux dépôts) et cliquez sur "Generate token" (Générer un jeton).
    Copiez le jeton généré (veillez à le copier avant de quitter cette page, car il ne sera plus affiché ultérieurement).

    Étape 4 : Configurez Git pour utiliser le jeton d'accès personnel

    Dans votre terminal Git, exécutez la commande suivante pour configurer le nom d'utilisateur :
  • git config --global user.name "Votre nom d'utilisateur"
    • Exécutez la commande suivante pour configurer l'adresse e-mail associée à votre compte GitHub :
  • git config --global user.email "votre@email.com"
  • git credential-manager-core configure
    • Si vous avez cette erreur :
    git : 'credential-manager-core' n'est pas une commande git. Voir 'git --help'.

    C'est qu'il semble que vous n'ayez pas le gestionnaire de credentials Git installé sur votre système.
